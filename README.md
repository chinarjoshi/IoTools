## Getting Started

### Prerequisites
Only for localhost usage

* pip
  ```sh
  pip install -r iotools/requirements.txt
  ```

### How to use

1. Clone the repo
   ```sh
   git clone https://github.com/chinarjoshi/iotools
   ```
2. Install prerequisite packages
   ```sh
   pip install -r iotools/requirements.txt
   ```
4. Run the flask entrypoint
   ```sh
   python3 iotools/src/wsgi.py
   ```
